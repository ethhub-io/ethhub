# Peer to Peer

For "Peer-to-Peer" decentralized exchanges, users are matched with one another off-chain for instant token swaps. Swaps are settled on-chain.

## Exchanges

* [AirSwap](airswap.md)
* [Ren](ren.md)

### Pros

* Convenient swapping of tokens 

### Cons

* No order system
